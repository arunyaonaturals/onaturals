# Import Instructions for Store Data

## 📤 Quick Import Steps

Your store data has been extracted and saved to: **`stores_data.csv`**

### Method 1: Using the Application's Import Feature

1. **Open the Application**
   - The app should already be open in your browser
   - If not, double-click `index.html`

2. **Navigate to Stores**
   - Click on **🏪 Stores** in the left sidebar

3. **Import CSV**
   - Click the **📤 Import CSV** button (top right)
   - Click **Choose File**
   - Select `stores_data.csv` from the Arunaya folder
   - Click **Import**

4. **Done!**
   - All 116 stores will be imported instantly
   - You'll see a success message
